# The Prime Resort Website

The Prime Resort Website is a responsive single-page website designed for a resort, focusing on clean navigation, clarity of information, and cross-device compatibility.

The project emphasizes structured layout and user-friendly design rather than complex backend functionality.

---

## Overview

The goal of this project was to design a visually clean and accessible website that presents resort information clearly across different screen sizes. Although implemented as a single-page application, the layout simulates a multi-page experience through structured navigation.

---

## Key Features

- Responsive single-page layout with structured navigation
- Mobile-first design ensuring compatibility across devices
- Clean and minimal UI focused on usability
- Consistent styling using reusable components and layout patterns

---

## Tech Stack

- **Frontend:** HTML, CSS
- **Styling & Layout:** Bootstrap
